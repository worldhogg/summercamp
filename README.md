### MyString

Собственная реализация строкового класса на C++17 — упрощённый аналог `std::string`.

Проект сделан для практического освоения управления ресурсами в современном C++:

- **RAII** — захват ресурса в конструкторе, освобождение в деструкторе
- **Правило пяти (Rule of Five)** — полная реализация: деструктор, копирующие и перемещающие конструктор и оператор присваивания
- **Move-семантика** — перемещение вместо копирования, `noexcept`-операции
- **Copy-and-swap идиома** — безопасное присваивание с защитой от самоприсваивания
- **Перегрузка операторов** — `[]`, `<<`, `==`, `!=`, `+`
- Методы: `size()`, `empty()`, `clear()`, `c_str()`

**Сборка:**
```bash
cd my_string
cmake -B build
cmake --build build
```

### Text RPG

Консольная ролевая игра — практика объектно-ориентированного проектирования.

Демонстрирует:

- **Иерархию наследования** в два уровня: абстрактный базовый класс `Character` → промежуточный абстрактный `Monster` → конкретные `Goblin`, `Dragon`, а также `Hero`
- **Полиморфизм** — боевая система работает с разными типами существ через указатели на базовый класс (`std::vector<Character*>`)
- **Абстрактные классы и чисто виртуальные функции** — единый интерфейс для всех персонажей
- **Виртуальный деструктор** — корректное освобождение ресурсов при удалении через базовый указатель
- **Разделение интерфейса и реализации** — объявления в `.h`, определения в `.cpp`

**Сборка:**
```bash
cd rpggame
g++ -std=c++17 -Iinclude src/*.cpp -o rpg
./rpg
```
